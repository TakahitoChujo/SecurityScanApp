プライバシーポリシー

最終更新日：2026年3月11日

Security Scan（以下「本アプリ」）は、スマートフォンの簡易セキュリティ診断ツールです。本アプリにおける個人情報の取り扱いについて、以下のとおりお知らせいたします。

1. 収集する情報

本アプリは、以下の情報を端末内でのみ利用します。外部サーバーへの送信は行いません。

  • カメラ映像：QRコードの読み取りに使用します。映像は保存・送信されません。
  • デバイス情報：OSバージョン、生体認証の有無などのセキュリティ診断に使用します。
  • ネットワーク情報：接続種別やIPアドレスを取得し、ネットワークリスクの判定に使用します。
  • スキャン履歴：読み取ったURLの履歴を端末のローカルストレージに保存します。

2. 外部サービスとの通信

QRコードで読み取ったURLの安全性を確認するため、Google Safe Browsing API にURLを送信する場合があります。この通信では個人を特定する情報は送信されません。

3. 個人情報の第三者提供

本アプリは、ユーザーの個人情報を第三者に提供・販売することはありません。

4. データの保存

すべてのデータは端末内のローカルストレージに保存され、アプリを削除すると完全に消去されます。アカウント登録やクラウド同期の機能はありません。

5. パスワードチェック機能

パスワード強度チェック機能で入力されたパスワードは、端末内でのみ処理されます。外部への送信は一切行いません。

6. 広告・アナリティクス

本アプリは広告の表示やアナリティクスツールの利用を行っていません。

7. お子様のプライバシー

本アプリは13歳未満のお子様から意図的に個人情報を収集することはありません。

8. ポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後のポリシーは本ページに掲載した時点で効力を生じるものとします。
